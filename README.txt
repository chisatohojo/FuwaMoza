ふわもざ / FuwaMoza
バージョン: v0.1.0

概要
----
ふわもざは、画像の一部にモザイクやぼかしを追加できる Windows 向けの軽量アプリです。
画像をドラッグ＆ドロップして、隠したい部分をクリックまたはドラッグでなぞるだけで加工できます。

主な機能
--------
- 画像のドラッグ＆ドロップ読み込み
- ファイル選択からの画像読み込み
- モザイク効果
- ぼかし効果
- クリックで円形加工
- ドラッグで線状加工
- ブラシ太さ調整
- 効果の強さ調整
- Undo
- ブラシ円プレビュー
- 元画像と同じサイズ・形式で保存
- 元画像を上書きしない安全な保存

対応形式
--------
- PNG
- JPG / JPEG
- BMP
- WEBP

使い方
------
1. FuwaMoza.exe を起動します。
2. 画像を画面中央へドラッグ＆ドロップします。
3. 効果を「モザイク」または「ぼかし」から選びます。
4. 太さと強さを調整します。
5. 隠したい場所をクリック、またはドラッグでなぞります。
6. 保存ボタンを押すと、元ファイル名に _fuwamoza が付いた画像が保存されます。

保存仕様
--------
- 元画像と同じフォルダに保存します。
- 元画像と同じ幅・高さで保存します。
- 元画像と同じ画像形式で保存します。
- 元画像は上書きしません。
- ファイル名は拡張子の前に _fuwamoza を付けます。
- 同名ファイルがある場合は _2, _3 のように連番を付けます。

保存例
------
sample.png -> sample_fuwamoza.png
sample_fuwamoza.png がすでにある場合 -> sample_fuwamoza_2.png

ショートカット
--------------
- Ctrl + O: 画像を開く
- Ctrl + S: 保存
- Ctrl + Z: Undo
- Esc: ドラッグ中の操作をキャンセル

使用ライブラリ
--------------
- PySide6
- Pillow
- PyInstaller

各ライブラリの利用条件は、それぞれのライセンスに従います。

注意事項
--------
- 元画像は上書きしません。
- 同名ファイルがある場合は連番を付けて保存します。
- 配布直後の exe は Windows の警告が出る場合があります。
- 不安な場合は、信頼できる配布元から入手したものだけを実行してください。
- 大切な画像は念のためバックアップを取ってから加工してください。

既知の制限
----------
- 現時点では矩形選択には対応していません。
- 大きな画像では処理に少し時間がかかる場合があります。
- EXIF 情報の完全保持は保証しません。
- アニメーション WEBP など複数フレーム画像は先頭フレームを対象にします。
